# WordPress Helper Tools

A collection of **plugins and scripts to optimize WordPress sites** for performance, security, and usability.

## Features

- Performance tweaks
- Security improvements
- Small utility plugins
- Easy integration with existing WordPress sites

## Installation

1. Download the plugin/scripts
2. Upload to `/wp-content/plugins/`
3. Activate in WordPress
